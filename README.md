# Aura Pipeline AI

Aura Pipeline AI is a premium AI social media content pipeline SaaS for creators, businesses, and marketing teams.

It helps users create brand workspaces, generate social media copy with Gemini-ready backend APIs, generate campaign posters, manage content history, plan scheduled posts, and export campaign assets.

## Highlights

- React, TypeScript, Vite, Tailwind CSS
- Node.js and Express API
- Email/password auth with protected dashboard routes
- Supabase-ready schema and auth support
- Gemini text/image generation with mock fallback mode
- Workspaces, content library, scheduler, analytics, and exports

## Local Development

```bash
npm install
npm run dev
```

Frontend: `http://localhost:5173`

API: `http://localhost:4000`
